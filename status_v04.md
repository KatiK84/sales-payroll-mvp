# Status — sales-payroll-mvp — v0.4

**Дата:** 18.03.2026
**Этап:** v0.4 — Редактор менеджеров в UI + динамический пересчёт

## Что сделано

### Новый блок: "Настройки менеджеров"
- Карточки менеджеров с inline-редактированием всех полей
- Поля: manager_name, режим_фикса, группа_фикса, тип_занятости, ручной_фиксированный_оклад, минимальный_порог
- При выборе режима "вручную" — поля группа/тип_занятости визуально отключаются
- При выборе режима "по группе" — поля ручного оклада/порога отключаются
- Кнопка "Сохранить" на каждой карточке + визуальный маркер "не сохранено"
- Кнопка "Удалить" с подтверждением
- Кнопка "+ Добавить менеджера"
- Кнопка "Пересчитать" перезапускает всю логику расчёта с текущим DIR

### Расчёт
- После сохранения карточки → пересчёт происходит автоматически
- После нажатия "Пересчитать" → обновляются все вкладки

### Редактор доступен двумя способами
1. На вкладке "Настройки менеджеров" (внутри панели после загрузки файла)
2. В отдельном блоке под формой загрузки (до загрузки файла)

## Поля расчёта
| Поле | Колонка |
|------|---------|
| Менеджер по продажам | col 4 |
| За товар без НДС (база фикса) | col 15 |
| За услуги без НДС (флаг услуги) | col 16 |
| Артикул (флаг EUPL) | col 7 |
