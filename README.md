- добавление `report_row_id`
- таблица mapping исходных и нормализованных колонок

## Как опубликовать
1. Загрузить файлы в репозиторий
2. Открыть **Settings → Pages**
3. Выбрать:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. Сохранить
5. Подождать, пока появится ссылка
