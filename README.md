# Sales Payroll MVP

Локальная тестовая версия программы расчёта зарплаты менеджеров по продажам.

## Стек
- Python
- Streamlit
- Pandas
- OpenPyXL

## Как запустить

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Что включено в этом архиве
Шаг 1:
- базовый каркас приложения;
- загрузка Excel-файла;
- предпросмотр данных.

Шаг 2:
- нормализация названий колонок;
- первичная очистка строк;
- удаление полностью пустых строк;
- попытка распознавания числовых колонок;
- добавление технического идентификатора строки.

## Важно
В архив намеренно НЕ включены скрытые файлы вроде `.gitkeep`,
чтобы его было удобно загружать в GitHub через веб-интерфейс.
