# Sales Payroll MVP — GitHub Pages Step 1

Статическая версия MVP, которую можно открыть через GitHub Pages.

## Что умеет
- загрузка Excel-файла прямо в браузере
- чтение первого листа
- показ количества строк и колонок
- вывод списка колонок
- предпросмотр первых 20 строк
- скачивание JSON предпросмотра

## Как опубликовать на GitHub Pages
1. Загрузить все файлы в репозиторий
2. Перейти в **Settings → Pages**
3. В разделе **Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
4. Нажать **Save**
5. Подождать 1–3 минуты
6. Открыть ссылку Pages

## Файлы
- `index.html`
- `style.css`
- `app.js`
